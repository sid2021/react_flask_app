# FLASK-REACT APP

Simple React App with Flask API backend.

## Built With

- [Flask](https://flask.palletsprojects.com/en/1.1.x/)
- [React](https://reactjs.org/)
- [yarn](https://yarnpkg.com/)

## Installation

```
$ git clone https://github.com/sid2021/flask_react_app.git
$ cd flask_react_app
$ npm install
$ npm install -g yarn
$ cd api
```

If you want to use virtualenv (on Windows):

```
$ python -m venv env
$ venv\scripts\activate
```

If you want to use virtualenv (on MacOS/Linux):

```
$ python3 -m venv venv
$ source venv/bin/activate
```

Install Python dependencies an run:

```
pip install -r requirements.txt
yarn start
yarn api-start
```

Go to `http://localhost:3000` (frontend).

Go to `http://localhost:5000/time` (backend).
